# بطاقة عرس إلكترونية

موقع عربي بسيط ومتجاوب لعرض بطاقة زفاف، جاهز للرفع على GitHub Pages.

## التعديل السريع

افتح `index.html` وعدّل:

- أسماء العروسين.
- تاريخ ووقت الزفاف.
- اسم القاعة والعنوان.
- رابط Google Maps.
- رقم واتساب داخل رابط `wa.me`.

ثم افتح `script.js` وعدّل بيانات حدث التقويم عند الحاجة.

## إضافة موسيقى

1. أضف ملفًا باسم `music.mp3` داخل المجلد.
2. في `index.html` احذف علامات التعليق حول سطر `<source>`.

## النشر على GitHub Pages

1. أنشئ مستودعًا جديدًا على GitHub.
2. ارفع الملفات الأربعة إليه.
3. افتح: `Settings` → `Pages`.
4. اختر `Deploy from a branch`.
5. اختر فرع `main` والمجلد `/root` ثم `Save`.
6. سيظهر رابط الموقع خلال وقت قصير في صفحة Pages.
